# Tools

This document describes the tools available to Ela.

## Built-in Tools

### Summarize
- Summarize web pages by URL
- Summarize PDF documents
- Summarize YouTube videos

## Future Tools

Additional tools can be added via plugins:
- Web search (oracle)
- Screenshot capture (peekaboo)
- Calendar integration (gogcli)
- And more...

## Usage Notes

- Always confirm before taking destructive actions
- Provide sources when summarizing content
- Ask for clarification if a request is ambiguous
